📡 SEO System

SEO Features
Per Artikel:
├── Dynamic metadata (generateMetadata)
├── Open Graph (title, description, image)
├── Twitter Card
├── Canonical URL
├── JSON-LD NewsArticle schema
└── allowIndex (robots meta)

Global:
├── /sitemap.xml          → all published posts, categories, tags, authors, pages
├── /news-sitemap.xml     → published posts (Google News)
├── /rss.xml              → latest published posts
├── /robots.txt           → disallow /admin, /api
└── Breadcrumb structured data (JSON-LD BreadcrumbList)

📐 Server Components Strategy

✅ Semua halaman publik pakai Server Components — fetch data langsung dari Payload REST API
✅ generateMetadata untuk SEO dinamis per halaman
✅ Client components hanya untuk interaksi: search input, mobile menu, share buttons
✅ ISR (Incremental Static Regeneration) atau SSR with cache untuk homepage & listing
✅ Data fetching via fetch dengan cache: 'force-cache' + Redis cache layer nanti
