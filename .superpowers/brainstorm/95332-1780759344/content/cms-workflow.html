🔄 CMS Workflow & Role System

Editorial flow + permission per role

Editorial Workflow
Author buat artikel (draft)
       │
       ▼
Author submit ke review
       │
       ▼
Editor review ──┬──→ Return to Draft (dengan catatan)
                │
                ▼
         Editor Publish / Schedule
                │
                ├──→ Published → Public Website
                │       │
                │       ├── Sitemap
                │       ├── News Sitemap
                │       ├── RSS Feed
                │       ├── Meilisearch Index
                │       └── Cache Invalidation
                │
                └──→ Scheduled → Otomatis publish di scheduledAt

👑 Admin

Full access — manage users, roles, all content, settings, clear cache, reindex search

✏️ Editor

Create/edit all articles, review, publish/schedule/archive, set featured & breaking news, manage categories/tags

📝 Author

Create own articles, edit own draft, upload media, submit to review. Cannot publish directly.

👀 Viewer

Read-only — view all data, no create/edit/delete

⚡ Slug & Redirect System

Slug auto-generate dari title, bisa diedit manual

Unique validation — tidak boleh ada slug duplicate

Redirect 301 otomatis saat slug post atau category berubah via Payload hook

Format: /[categorySlug]/[postSlug]
